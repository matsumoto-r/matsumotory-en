## MATSUMOTO Ryosuke

[[English]](http://research-en.matsumoto-r.jp/) [[Japanese]](http://research.matsumoto-r.jp/)

After getting bachelor, he worked for three years in a hosting company. Experience in the company was accepted, and he skip the master's program and admmition to doctoral program in the Graduate School of Informatics, Kyoto University. He has been researched about OS, Middleware and protocol. He interested in security, resource management, performance, large-scale, operation technology for Web servers, the next generation infrastructure technology like Test-Driven Infrastructure and Immutable Infrastructure, and the next generation HTTP protocol like SPDY, HTTP/2 and QUIC in particular.
